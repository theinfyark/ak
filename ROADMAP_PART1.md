# Reskilling & Job-Hunt Roadmap (9+ YOE)

**Path:** Azure → Docker → Kubernetes → Terraform → Node.js → React → Next.js → Traditional AI/ML → Vector DBs → Gen AI/RAG

**Rule:** One topic at a time, in order. Don’t start topic 2 until topic 1’s checklist is done. Within each topic, learn sub-topics in sequence (earlier items are prerequisites for later ones).

---

## Part 1 — High-Level Sequence Checklist

| Week | Topic | Daily Pace | Checklist |
|------|-------|------------|-----------|
| **1–3** | Azure Fundamentals | 1–1.5 hrs/day | ☐ Concepts done<br>☐ AZ-900 exam booked<br>☐ AZ-900 passed |
| **4–6** | Docker | 1–1.5 hrs/day | ☐ Concepts + hands-on done<br>☐ Can containerize an app |
| **7–9** | Kubernetes | 1.5 hrs/day | ☐ Concepts + hands-on done<br>☐ Can deploy an app to a local cluster (minikube/kind) |
| **10–12** | Terraform | 1.5 hrs/day | ☐ Concepts + hands-on done<br>☐ Terraform Associate exam booked<br>☐ Terraform Associate 004 passed |
| **13–15** | Node.js | 1–1.5 hrs/day | ☐ Concepts done<br>☐ One API built end to end |
| **16–17** | React | 1–1.5 hrs/day | ☐ Concepts done<br>☐ One app built end to end |
| **18** | Next.js | 1.5 hrs/day | ☐ Concepts done<br>☐ App migrated/rebuilt in Next.js |
| **19–20** | Traditional AI/ML | 1–1.5 hrs/day | ☐ Concepts done (fluency only — no coding required) |
| **21** | Vector Databases | 1 hr/day | ☐ Concepts done<br>☐ One small embedding search demo built |
| **22–23** | Gen AI / RAG | 1.5–2 hrs/day | ☐ Concepts done<br>☐ Mini RAG chatbot built<br>☐ AI-901 exam booked<br>☐ AI-901 passed |
| **24–25** | Integration Project | 2 hrs/day | ☐ Next.js + Node API + RAG chatbot built<br>☐ Containerized with Docker<br>☐ Deployed to Azure (AKS or App Service) via Terraform |

**~25 weeks (~6 months)** at a steady pace. If you already have strong exposure in one area, compress that section and bank the time for weaker areas — don’t skip the sequence, just move faster through what you know.
